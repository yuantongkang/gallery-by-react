### 项目描述
使用 react-webpack 脚手架制作一个图片画廊

### 技术栈
+ react： 当下三大前端主流框架之一

+ webpack： 当下最流行的前端打包工具

+ scss： 高效快捷的预编译 CSS

+ autoprefixer： 自动添加 CSS 浏览器兼容前缀

+ es6 语法： 更加简洁高效


### 环境搭建

```
npm install -g yo

npm install -g generator-react-webpack

```
### 项目构建

```
yo react-webpack
```

### 项目启动
```
git clone git@github.com:18810666595/gallery-by-react.git

cd gallery-by-react

npm run serve

```
